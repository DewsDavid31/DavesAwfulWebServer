Download tar from following link-
https://dev.mysql.com/get/Downloads/MySQL-8.0/mysql-8.0.29.tar.gz


After compiling mysql go through following steps 

- /Applications/AMPPS/apps/mysql/bin/mysqld --initialize --basedir=/Applications/AMPPS/apps/mysql --datadir=/Applications/AMPPS/apps/mysql/data

start the mysql service before changing the password to start the service (/Applications/AMPPS/apps/mysql/bin/mysqld)

- /Applications/AMPPS/apps/mysql/bin/mysql -u root -p (enter temp password)

- ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'mysql'; (change password)

- /Applications/AMPPS/apps/mysql/bin/mysqladmin --user=root --password shutdown
