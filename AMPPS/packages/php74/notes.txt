WAEMP.sh functions to compile PHP 74 for AMPPS Mac OS
- php74_cgi
- mod_php74
- php74
- php74_libsodium
- mongo74

Make sure libphp7.so is moved from $PATH_APACHE2/modules/libphp7.so $PATH_PHP74/lib/

Make sure following Extension are created in $PATH_PHP74/ext
- sodium.so (not working)
- mongodb.so

Run following command after compilation(To change zstd library default path in mongodb.so)
sudo install_name_tool -change /usr/local/lib/libzstd.1.dylib /Applications/AMPPS/apps/lib/libzstd.1.dylib /Applications/AMPPS/apps/php74/ext/mongodb.so
